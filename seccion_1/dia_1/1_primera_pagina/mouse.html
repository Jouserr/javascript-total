eventoclick: El evento se ejecuta cuando el usuario hace click con el botón izquierdo del ratón


evento onmousedown: El evento se ejecuta cuando el usuario hace click con el botón izquierdo o derecho del ratón


evento onmouseup: El evento se ejecuta cuando el usuario libera el botón izquierdo del ratón


evento onmouseenter: Este evento se desencadena cuando el ratón se introduce en el área ocupada por el elemento.


evento onmouseleave: Este evento se desencadena cuando el ratón abandona el área ocupada por el elemento.


evento onmouseover: . Este evento se desencadena cuando el ratón se mueve sobre el elemento o cualquiera de sus elementos hijos.


evento onmouseout: El evento se desencadena cuando el ratón abandona el área ocupada por el elemento o cualquiera de sus elementos hijos.


evento onmousemove: Este evento se desencadena cada vez que el ratón se encuentra sobre el elemento y se mueve.


evento onwheel: Este evento se desencadena cada vez que se hace girar la rueda del ratón.
